## An interactive dashboard for customer analysis and churn prediction using machine learning

---

## ✨ About The Project

A web-based interactive application that helps businesses understand customer behavior through:

- **Churn Prediction** - Predict whether a customer will stay or leave
- **Customer Segmentation** - Group customers into 5 clusters based on income and spending
- **Data Visualization** - Interactive histograms and scatter plots

---

## 🚀 Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/MohamedAbdeenM7/Customer-Analytics-Dashboard.git
cd customer-analytics
```

### 2.Install Requirements

```bash
pip install -r requirements.txt
```

### 3. Run the App

```bash
streamlit run app.py
```

### 🛠️ Built With

- Python - Core programming language
- Streamlit - Interactive web interface
- Scikit-learn - Machine learning models
- Pandas - Data manipulation
- Matplotlib - Data visualization

### 📊 Datasets

- churn_dataset.xlsx: Customer data for churn prediction
- Mall_Customers.csv: Customer data for segmentation (income, spending score, age)
